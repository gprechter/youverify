class State.State¶

Abstract class template for a representation of State in YouVerify.

State.State.next_state(self)¶

Returns TRUE if there is more execution to be done, FALSE otherwise.

This method is invoked by the framework when before the next statement is to be executed. This gives the state the opportunity to modify the backend and setup the State object for the next statement execution. A developer would modify this to change how the paths of a program are explored and how state is handled.

Returns

True if there is more state to be executed, False otherwise.

Return type

bool
